一、 高階主管戰略摘要 (Executive Summary)

本篇研報依據「福爾摩沙 AI (FALO AI)」之內部知識庫、系統標準作業流程 (SOP)、常見問答 (FAQ) 以及專案時程規範進行全面盤點與深度解析。本公司專注於核心業務與技術研發，主要重點服務項目包含「企業級智慧 Agent 開發」、「多模態偵查系統建置」、「知識庫系統開發」以及「企業資源規劃系統 (FALO ERP) 之部署、系統對接與權限設定」[依據：📄 全局企業背景提示詞與 FALO ERP 系統標準作業流程]。

在公司營運與專案推展方面，本公司提供完善的企業級服務，包含 7 天滿意鑑賞期與 7x24 企業級維護合約，並嚴格遵循最高企業防幻覺 Guardrails 規範與 100% 台灣繁體中文標準 [依據：📄 全局企業背景提示詞]。此外，115 年度的重要里程碑正積極推進 ERP 系統的各階段開發，包含需求訪談、API 對接、測試驗收及 10/01 正式上線 (Go-Live) [依據：📄 115 年度 FALO ERP 專案時程與重要里程碑]。

二、 內部規範與歷史發言詳盡解析 (Internal Insights)

1. 核心技術與系統開發服務

FALO AI 作為專業的企業級解決方案供應商，其核心重點服務項目涵蓋以下幾個維度：

  • 企業級智慧 Agent 與多模態偵查系統：專注於為企業客戶打造高度智慧化的代理與多模態偵查工具，協助企業進行自動化流程與深度資料分析 [依據：📄 全局企業背景提示詞]。
  • 知識庫系統開發：建置高效率的企業內部知識庫管理系統，確保企業內部資訊的正確傳遞與高標準防幻覺機制 [依據：📄 全局企業背景提示詞]。
  • FALO ERP 系統建置與部署：提供完整的 ERP 系統建置服務，並規範完善的權限指派與系統維護流程 [依據：📄 FALO ERP 系統部署與權限設定標準作業流程]。

2. ERP 系統權限與角色規範

在 FALO ERP 重點服務系統中，內部權限管理分為三大角色，確保企業營運資安與流程順暢：

  • 系統管理員 (Admin)：具備最高權限，負責修改系統設定、備份資料庫 [依據：📄 FALO ERP 系統部署與權限設定標準作業流程 §1]。
  • 一般操作員 (User)：負責日常單據輸入與查詢報表 [依據：📄 FALO ERP 系統部署與權限設定標準作業流程 §1]。
  • 主管簽核 (Manager)：具備請購單與出貨單之核准權限 [依據：📄 FALO ERP 系統部署與權限設定標準作業流程 §1]。

3. 系統維護、備份與專案分工規範

  • 資料庫備份：系統每日凌晨 02:00 自動執行完整備份 (Full Backup) 並存於 AWS S3；系統管理員須於每週五下午 17:00 執行人工異地備份 [依據：📄 FALO ERP 系統部署與權限設定標準作業流程 §2]。
  • 專案時程與分工：115 年度專案重點包含 API 對接（由研發部 Jerry 負責）、資料庫備份規則制定（由主管 鄭Force 負責督導）以及教育訓練手冊編撰（由 FALO 服務窗口 負責）[依據：📄 115 年度 FALO ERP 專案時程與重要里程碑]。
